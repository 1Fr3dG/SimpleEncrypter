## Installation

可通过[CocoaPods](http://cocoapods.org)安装：

SimpleEncrypter is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod "SimpleEncrypter"
```
